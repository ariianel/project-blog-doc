## What is WLKOM?

WLKOM is a **Linux Kernel Module (LKM)** acting as a rootkit. Once loaded on a victim machine, it connects to a C2 server, polls for commands, and hides itself from the system.

<CardGrid>
  <Card title="C2 Server" icon="laptop">
    A Flask web application running on the attacker's machine. Provides a real-time dashboard to manage victim machines, dispatch commands, and transfer files.
  </Card>
  <Card title="Rootkit (LKM)" icon="puzzle">
    A Linux Kernel Module loaded on the victim machine. Polls the C2 every 5 seconds, executes commands, transfers files, and hides itself from userland tools.
  </Card>
</CardGrid>
