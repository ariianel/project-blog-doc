## What is WLKOM?

WLKOM is a **Linux Kernel Module (LKM)** acting as a rootkit. Once loaded on a victim machine, it silently connects to a command-and-control (C2) server and awaits instructions — all while hiding itself from the system.

<CardGrid>
  <Card title="C2 Server" icon="laptop">
    A Flask web application running on the attacker's machine. Provides a real-time dashboard to manage victim machines, dispatch commands, and transfer files.

    **Python · Flask · SQLite · AJAX**
  </Card>
  <Card title="Rootkit (LKM)" icon="puzzle">
    A Linux Kernel Module loaded on the victim machine. Polls the C2 every 5 seconds, executes commands in kernel space, transfers files, and hides itself from userland tools.

    **C · Linux Kernel · HTTP/1.0 · Syscall hooks**
  </Card>
